# KamisasFlix 🎬🍿
API simples desenvolvida para gerenciar informações sobre filmes e séries, 
permitindo realizar operações de cadastro e pesquisa com filtros.

## Tecnologias utilizadas
- Node.js
- Express
- JavaScript
- JSON

## Endpoits da API

### Filmes 🎥
| Método | Endpoint | Descrição |
|--------|----------|-----------|
| GET | /filmes | Lista todos os filmes |
| GET | /filmes/:id | Busca de um filme pelo seu id |
| POST | /filmes | Cria um Novo filme |


### Séries 🎞️
| Método | Endpoint | Descrição |
|--------|----------|-----------|
| GET | /series | Lista todas as séries |
| GET | /series/:id | Busca de uma serie pelo seu id |
| POST | /series | Cria uma nova serie |
